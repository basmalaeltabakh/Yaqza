# Yaqza | يقظه 
Yaqza is an end-to-end Predictive Maintenance system that leverages Industrial IoT sensor streams and Machine Learning to predict equipment failures before they occur.
Unlike traditional anomaly detection that simply flags unusual readings, Yaqza predicts the Remaining Useful Life (RUL) of industrial machinery — giving factory operators a 48–72 hour maintenance window to act before failure happens.
